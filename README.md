# Image Gallery App
This is a simple and responsive Image Gallery App built using HTML and CSS, which displays images in a clean grid layout with hover effects.

## Features
* Fully responsive design
* Grid-based image layout
* Smooth hover effects
* Easy to customize

## Technologies
* HTML5
* CSS3

## Project Structure
**act6_my_gallery-main**
* images
* index.html
* style.css
* README.md
* LICENSE

## Getting Started
1. Clone the repository git clone: https://github.com/AL-FAIZKHAN007/act6_my_gallery.git
2. Open the project & double click on index.html file
